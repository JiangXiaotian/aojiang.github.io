  1. 什么是机器学习？
    想让计算机像人一样思考所研发出来的计算机理论。
    人和计算机其实本没有差别，同样都是一大批互相连接的信息传递和存储元素所组成的系统。
    包括概率论、统计学等等学科。
    Google Now, Google Photo

  2. 机器学习算法
    目前所有的机器学习算法大概可以被分为 4 到 5 类，
    监督学习(数据和标签)
    非监督学习（不提供标签的样本，通过观察各种数据之间的特性，会发现这些特性背后的规律，这些规律也就是非监督学习所学到的东西）
    半监督学习（少量有标签的样本和大量没有标签的样本）
    强化学习（规划机器人准则方面，把计算机丢到一个对于它完全陌生的环境，让它完成一项从未接触过的任务,它会自己去尝试各种手段，最后让自己成功适应这一个陌生的环境,从经验中总结提升）
    遗传算法（模拟进化理论，淘汰弱者，适者生存，马里奥一代->随机生存二代->保存最厉害的马里奥,淘汰弱的->基于强者“繁衍和变异”->更强马里奥）

  3. 人工神经网络 VS 生物神经网络
    900 亿的神经网络细胞，通过刺激产生新连接
    人工神经网络连接固定，不会凭空产生,靠正向和反向传播更新神经元
    特征->代表特征->代表特征->

  4. 神经网络
    每个神经元都有属于它的刺激函数，用这些函数给计算机刺激行为

  1. 卷积神经网络
    不是对每个像素的输入做处理，对图片每一小块区域的像素做处理，加强图片信息的连续性，加深计算机对图片的理解，批量过滤器件，持续不断在图片上滚动，每一次搜集到一小块的图片信息，对搜集到的信息整理，产生一些边缘信息，再用图片过滤器扫描这些边缘信息，总结出更高层的信息结构，再经过一次过滤，最后把这些信息套入普通的全连接神经网络进行分类，
    在每一次卷积的时候，神经元会不经意丢失一些信息，池化（pooling）可以很好解决，卷积的时候不压缩长宽，尽量保留更多信息，压缩的工作交给池化。
    image -> convolution -> pooling -> convolution -> pooling -> fully connected -> fully connected -> classifier

  2. RNN(循环神经网络)
    记住发生事情的能力，把之前的记忆一起分析

  3. LSTM(长短期记忆) RNN
    在反向传递得到的误差的时候, 他在每一步都会乘以一个自己的参数 W. 如果这个 W 是一个小于 1 的数, 比如 0.9. 这个 0.9 不断乘以误差, 误差传到初始时间点也会是一个接近于零的数, 所以对于初始时刻, 误差相当于就消失了. 我们把这个问题叫做梯度消失或者梯度弥散 Gradient vanishing. 反之如果 W 是一个大于1 的数, 比如 1.1 不断累乘, 则到最后变成了无穷大的数, RNN被这无穷大的数撑死了, 这种情况我们叫做剃度爆炸, Gradient exploding. 这就是普通 RNN 没有办法回忆起久远记忆的原因.
    多出输入、输出、忘记控制器
    主线剧情、分线剧情

  4. 自编码(Autoencoder)
    非监督学习
    图片 -> 压缩（原数据的精髓） -> 解压

  5. 生成对抗网络(GAN) Generative Adversarial Nets
    Generator 、 Discriminator

  6. 优化方法
    Newton’s method、Least Square method、Gradient Descent
    cost = (predicted - real)^2 = (Wx - y)^2

  7. 过拟合
    增加数据量
    L1/L2 正规化
    drop out

  8. 标准化
    minmax normalization (0,1) 或者 (-1,1)
    std(standard deviation) normalization(mean=0, std=1)

  9. Good feature
    不好的特征混淆感官, 浪费我们的分析, 计算资源
    避免无意义、重复性、复杂的信息

  10. 激励函数
    解决线性方程不能概括的问题
    relu、sigmoid、tanh
    激励函数必须是可微分的
    少量层
    CNN 推荐 relu
    RNN 推荐 relu or tanh

  11. 加速神经网络训练
    Stochastic Gradient Descent(SGD): 使用批量数据,加速训练过程, 也不会损失太多准确率
    大多在更新神经网络参数的时候动动手脚
    W += -Learning rate dx
    momentum:
    m = b1    m - Learning rate dx
    W += m
    Adagrad:
    v += dx^2
    W += -Learning rate     dx / v^(1/2)
    RMSProp:
    m = b1m + (1-b1)dx
    v = b2v + (1-b2)dx^2
    W += -Learning rate * m / v^(1/2)

  12. 不均衡数据
    想办法获取更多的数据
    换个评判方式(Confusion Matrix -> Precision & Recall -> F1 Score)
    重组数据(复制合成少数部分，砍掉多数部分)
    使用其他机器学习方法
    修改算法

  13. 批标准化
    分散数据统一规格
    batch normalization

x -> 全连接层 -> batch normalization -> 激励函数 -> 全连接层。。。

  1. 强化学习

分数导向性

Q-learning
Sarsa
Deep Q Network

Policy Gradient

Model based RL

分类：

  1. model-free or model-based
  2. Policy-Based RL(基于概率, 连续动作) or Value-Based RL
    Actor-Critic(Actor 基于概率给出动作, Critic 对于动作给出价值)
  3. Monte-Carlo update 回合更新和 Temporal-Difference update 单步更新

  4. 在线学习(Sarsa、Sarsa(r))和离线学习(不必是自己经验)

  5. DQN
    Experience replay, 打断经历之间的相关性，使神经网络的更新更有效率
    Fixed Q-target(Q现实用的很久以前的参数，Q估计用的最新参数)

  6. Policy Gradients
    连续动作
    通过奖励调整选择某一动作的概率
    回合更新

  7. Actor Critic
    当在连续状态中更新，每次参数更新前后有相关性, 导致神经网络只能片面看待问题, 甚至学不到东西。

  8. DDPG(Deep Deterministic Policy Gradient)
    Actor Critic + DQN
    在连续动作上更有效的学习

  9. Asynchronous Advantage Actor-Critic (A3C)
    一种有效利用计算资源, 并且能提升训练效用的算法
    平行训练
